# Context Switching Simulation (Operating System)

## Description
This project is a web-based simulation of Context Switching in Operating Systems. It visually demonstrates how the CPU switches between multiple processes.

## Features
- Process switching visualization
- CPU scheduling concept demonstration
- Simple interactive UI

## Technologies Used
- HTML
- CSS
- JavaScript

## Purpose
To understand Operating System concepts like process management and context switching through visualization.
