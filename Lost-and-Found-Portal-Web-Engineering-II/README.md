# Lost & Found Portal - Web Engineering II

## Description
A web-based Lost & Found portal where users can report lost items or post found items and search through listings.

## Features
- Post lost or found items
- Search functionality
- User-friendly interface
- Basic data handling system

## Technologies Used
- HTML
- CSS
- JavaScript
- (Optional backend if used)

## Purpose
To develop a real-world web application and understand web engineering concepts.
