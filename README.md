# Semester 5 Projects

This repository contains projects completed during Semester 5 of my Bachelor of Science in Information Engineering Technology degree.

## Projects

### 1. Context Switching (Operating System)
- Web-based simulation of context switching in Operating Systems
- Helps visualize process switching and CPU scheduling concepts
- Built using web technologies for better understanding of OS concepts

### 2. Lost & Found Portal (Web Engineering II)
- A web-based portal for reporting and finding lost or found items
- Allows users to post and search items
- Includes basic authentication and data management features

### 3. Blockchain for Networking: Secure Transactions Simulation (DCCN)
- A basic simulation of blockchain technology
- Demonstrates secure transactions in IoT/mobile network environments
- Focuses on decentralized and secure communication concepts

## Technologies Used
- HTML, CSS, JavaScript
- Web Engineering Concepts
- Basic Backend 
- Simulation-based logic

## Purpose
To implement advanced computing concepts such as Operating Systems, Web Engineering, and Blockchain through practical and simulation-based projects.
