# Blockchain for Networking: Secure Transactions Simulation (DCCN)

## Description
This project is a basic simulation of blockchain technology used for secure transactions in IoT and mobile network environments.

## Features
- Blockchain block creation simulation
- Secure transaction flow
- Visualization of blockchain structure

## Technologies Used
- Blockchain concepts
- Networking fundamentals

## Purpose
To understand blockchain technology and its application in secure networking systems.
